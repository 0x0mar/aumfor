2.How to install AUMFOR

• Linux

◦ Open a terminal.

◦ Change to setup's directory:

▪ cd “/path/to/setup”

◦ Run the file “setup.pyc” with sudo:

▪ sudo python setup.pyc install

▪ Note that you'll be asked to enter your account password:

• To launch, open a terminal and use “aumfor” command.

• Windows

◦ Launch Command Propmpt as Adminstrator. You may be asked to enter administrator's password:

◦ Change to setup's directory:

▪ cd “\path\to\setup”

◦ Run the file “setup.pyc” with sudo:

▪ sudo python setup.pyc install

• To launch, find the icon on Desktop

• Double click on AUMFOR Server Icon to start Server

• Double click on AUMFOR Icon to start using AUMFOR in Web-browser

◦ Registry Hives

A hive is a logical group of keys, subkeys, and values in the registry that has a set of supporting files containing backups of its data.

Each time a new user logs on to a computer, a new hive is created for that user with a separate file for the user profile. This is called the user profile hive. A user's hive contains specific registry information pertaining to the user's application settings, desktop, environment, network connections, and printers. User profile hives are located under the HKEY_USERS key.
